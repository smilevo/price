Dataset

Description

The datasets listed on this page are the calculated metrics for the commits under study. Dataset 1 contains the light-weight dynamic metrics. The second dataset contains the light-weight dynamic metrics along with 35 static structural metrics listed in the original paper. The last dataset named non-correlated, contains the light-weight performance metrics along with the non-correlated static structural metrics.

Dataset 1

Includes the the light-weight dynamic metrics

Dataset 2

Includes the the light-weight dynamic metrics along with 35 static structural metrics.

Dataset 3

Includes the the light-weight dynamic metrics along with the non-correlated static structural metrics.
